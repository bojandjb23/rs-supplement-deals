Implementation Roadmap

Recommended implementation order

P1
Deal card redesign
Highest impact on conversion. Full-width CTA + prominent savings chip. Update createProductCard() in supplement-dashboard.js.
P2
Category pills with icons
Drop-in replacement for current text tabs. Update buildCategoryTabs() to use pill template with Phosphor icons.
P3
Email capture banner on dashboard
Insert static HTML between stats bar and product grid. Wire dismiss to localStorage. No JS changes needed to existing dashboard logic.
P4
Mobile bottom navigation
Replace floating FAB. Add hidden lg:hidden bottom nav bar. Add pb-[76px] lg:pb-0 to main content container to avoid overlap.
P5
Header restructure (3-row)
Most invasive change — requires restructuring the sticky header HTML. Recommend doing last after other changes are validated.
